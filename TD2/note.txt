Script perl :
 - S'il existe motLexique = motRequète -> lemme
 - S'il existe motLexique proche motRequète -> 	préfixe (grandeur proche de grand)
 											->	édition orthographe (grenade proche de grande)
Fonction java :
paramètre : mot que l'on cherche
retour :
 - retrouver le mot du lexique, et retourner le lemme
 - chercher les préfixes les plus proches et retourner la liste des lemmes
 - chercher les mots clés les plus proches et les retourner sous forme de liste
 - sinon retourner le mot entré en paramètre (mot trop mal orthographié...)
